HTTP Server in C++

This is a simple HTTP server implemented in C++. It listens for incoming connections and serves basic HTML content based on the requested path.

To run the server, compile the code and execute the resulting binary. The server will start listening on port 8080. You can access it by navigating to http://localhost:8080 in your web browser.

The server supports basic HTTP methods such as GET, HEAD, POST, and PUT. Depending on the requested path, it will return different HTML content. For example, accessing the root path ("/") will display a home page, while accessing "/about" will show an about page.
